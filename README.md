<h1>To run the file carry out the following steps</h1>
<p>Make sure that mongoDb is running in the background for database</p>
<div>
  <p>For the Client Folder</p>
  <ol>
    <li>Change the file path to client after opening the folder</li>
    <p>cd client</p>
    <li>Then run npm install for dependencies</li>
  </ol>
  <p>for the Server folder</p>
  <ol>
    <li>change the file directory to cd server</li>
    <li>then run the command "npm run dev"</li>
  </ol>
  
</div>
